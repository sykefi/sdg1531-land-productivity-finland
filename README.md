# EOSDG WP2 demonstration tools

This repository contains example workflow for nationally adapted SDG 15.3.1 subindicator calculation from custom Landsat timeseries dataset.

Overall descption is given in [README-win.md](README-win.md) and a example executable workflow in [Workflow-example.md](Workflow-example.md)
